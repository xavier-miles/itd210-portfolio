# itd210-portfolio
Repository for the ITD 210 project files
Xavier Miles
The topic of my portfolio is my animal care business I run in my neighborhood. There will be pages for gallery, about me, services provided, and a form for requesting a booking.

Project done for ITD 210: Web Page Design II in 2026 Spring semester

Using HTML5, CSS3, JS, and Tailwind
Image compression done with https://www.imgonline.com.ua/eng/compress-image-size.php

AI-assisted code is labelled and explained

(Due 4/20/2026) Milestone 3: Page 3
Worked on the Contact page, implementing a responsive form with subtle interactivity and integrated with Formspree for submission functionality.

(Due 4/27/2026) Milestone 4: Page 4
Worked on Services page, adding an introductory video, background SFX, and a table laying out all my options. Booking options have buttons to send you to the contact form with an option pre-selected.

(Due 5/4/2026) Milestone 5: Page 5
Worked on Testimonials page, adding pictures with expandable quotes attached. Bonus interactivity on comfortably large screens.
Also made tweaks to other pages, including adding client-side validation for contact.html and tidying things up globally.

(Due 5/10/2026) Milestone 6: Final Deployment
Did a major sweep-through of the full site, tightening up spacing throughout and improving consistency of the display and feel of things.
Added site-wide skip link as well to enhance accessibility. I made some changes to the design of about.html as well. Final submission for ITD 210!


Final reflection:
I would say the strongest part of my site is the Gallery page. Not only does it demonstrate some familiarity with Tailwind CSS, which is an industry-standard CSS framework, I am proud that I successfully implemented the lightbox.
In terms of progressing my learning of responsive web design and interactivity, I feel much more capable of writing things to be mobile-friendly from the start, instead of building it for desktop first and then working in responsive layouts. I also feel more confident in my ability to design intentional and meaningfully interactive features.
The feedback I received about working to make the site more consistent and tighten spacing helped a lot. It pushed me to take a look at the website and think, "How can I make this feel like one cohesive, intentional site?"
If I had another month, I would work on making it more flashy. Where it stands, I believe it to be functional and not displeasing, but most of it does not particularly catch the eye or make you go "Wow!"
